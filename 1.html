Hypertext Markup Language(HTML) is the standard Markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and oringally included cues for the appearance pf the doument.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semanics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, wriiten using angle brackets.
